Product Vision:

Project Huddle is an agile project management application that helps scrum teams keep track of projects. In agile project management User Stories are descriptions of project requrements broken down into manageable parts that help the team plan their project. This tool tracks lists of user stories and estimated time it takes to complete each story. Actual time spent can also be tracked. The time is presented in a burndown chart to help the team know if their goals are being reached or if adjustments need to be made.
